# Multiplayer RPG 3D
Unity Version: **2019.3.2f1**
Networking: **Mirror Networking**
Features:
* Saving player's data to SQLite database  
* Entities controlled by server  
* Inventory & Equipment system
* Player state synchronization
* Item pickup  
* Fight system  
* Draggable UI windows
